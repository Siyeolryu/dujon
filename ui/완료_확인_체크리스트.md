## 수정·추가된 파일 목록

- `site-management.html` — 대시보드 차트 영역, 테이블 구조, 스켈레톤/빈 상태 영역
- `css/style.css` — 비주얼 전면 개편, 테이블, 반응형
- `css/components.css` — 스켈레톤 UI 스타일
- `css/assign-panel.css` — 색/테두리 정리
- `js/app.js` — 테이블 렌더링, 스켈레톤, 선택 행 갱신
- `js/dashboard.js` — 스켈레톤, SVG 차트
- `js/api.js` — assign/update/unassign 시 If-Match 헤더
- `js/site-detail.js` — 상세 열릴 때 `App.updateSelectedRow(siteId)` 호출
- `js/config.js` — 상태/뱃지 색상 파스텔 톤

## 결론

**ui/ux-디자이너.md 가이드라인 기준으로 요구된 항목은 모두 반영·구현된 상태입니다.**

실행 확인 방법:
1. API 서버 실행: `python run_api.py` (또는 `flask run`)
2. 브라우저에서 `site-management.html` 열기
3. 초기 로딩 시 스켈레톤 → 통계·차트·테이블 표시, 행 클릭 시 상세 패널 슬라이드 인, 배정/해제 시 If-Match·version 전달 확인
