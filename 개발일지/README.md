## 작성 규칙

1. **파일명 형식**: `개발일지_날짜_제목.md`

   - 제목: 해당 일지의 주제 (한글/영문 가능)
2. 같은 날 여러 일지를 작성할 경우, 제목만 다르게 하면 됩니다.
   - 예: `개발일지_2026-01-30_프로젝트현황.md`, `개발일지_2026-01-30_API개발진행.md`
