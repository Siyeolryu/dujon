# 테스트 데이터 업로드 안내

## 파일

- **테스트_현장_5개.csv** — 현장 5건 (T001~T005)
- **테스트_소장_5인.csv** — 소장 5인 (P-T01~P-T05)

## Google 스프레드시트에 넣는 방법

1. [DB 스프레드시트](https://docs.google.com/spreadsheets/d/15fAEzkC9FCLA6sG1N--f69r-32WHoYLvmXcwED5xWzM/edit) 열기

2. **기존 데이터 지우기**
   - **현장정보** 시트: 데이터 행 전체 선택(헤더 제외) → 삭제
   - **인력풀** 시트: 데이터 행 전체 선택(헤더 제외) → 삭제
   - (자격증풀은 그대로 두거나 필요 시 따로 처리)

3. **현장 5개 넣기**
   - **현장정보** 시트 선택
   - 메뉴: **파일** → **가져오기** → **업로드** 탭
   - `테스트_현장_5개.csv` 선택
   - **가져오기 위치**: "현재 시트에 삽입" 또는 "현재 시트 바꾸기" (기존 헤더가 있으면 "데이터만 가져오기" 등으로 2행부터 삽입)
   - **구분 기호**: 쉼표
   - **텍스트를 숫자·날짜로 변환**: 원하면 사용
   - **데이터 가져오기** 클릭

4. **소장 5인 넣기**
   - **인력풀** 시트 선택
   - 메뉴: **파일** → **가져오기** → **업로드**
   - `테스트_소장_5인.csv` 선택
   - 위와 같이 가져오기 옵션 선택 후 **데이터 가져오기**

5. **또는 복사·붙여넣기**
   - CSV를 메모장 등으로 열어 전체 복사
   - 시트에서 헤더 다음 셀(A2) 선택 후 붙여넣기

## 인코딩

- CSV는 **UTF-8** 저장. 한글이 깨지면 Google 시트 가져오기 시 인코딩을 UTF-8로 선택하거나, 메모장에서 "다른 이름으로 저장" → 인코딩 **UTF-8**로 다시 저장 후 업로드.
